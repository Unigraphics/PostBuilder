Listing File & Output Control

The listing dialog will allow you to control how the listing file will be formatted.  You can also define how your post will handle groups, warnings and the review tool.  You can also define what the default extension will be on your NC output file and your listing file.
 

List of Parameters

You can use this dialog to disable or enable the output of various information on your listing file.  Only those items with a check mark will be output.
 

Output Control Elements

Generate Group Output

This option will allow you to define how your post will handle groups.  You can organize your operations into groups in Unigraphics to create more than one NC program in a part file.  Each group is considered to be a program and should be considered one complete NC tape.

The default setting for this option is OFF.  You should not use this option unless you want to postprocess more than one NC program at a time.

If you select more than one group and you have this option turned ON you will get one file that contains all of the groups (programs) that you have selected.  This file will be named my_file.ptp.  You will also get an NC program file for each group or program you have selected.  They will be named my_file_group.ptp.  If your base file name is "nctape" and your group or program name is "O1234" then the name of your file will be nctape_O1234.ptp.

When using the group output is ON, you should not mix operations in groups with operations outside of groups.  This will cause unpredictable output.

When the group output setting is OFF, you will only get a single NC output file called my_file.ptp.  All operations in any selected group will be output in a single NC file.  All group organization will be ignored.

The name of the group or program in Unigraphics can be used as the program name in the Start of Program event.  This will be passed to you as the variable name mom_group_name.
 

Output Warning Messages

When this option is set to ON, warnings will be output to a warning log file.  The name of this file will be named my_file_warning.out and will reside in the same directory that your NC output is located.
 

Activate Review Tool

When this option is set to ON, an analysis program will be executed to help you review the output of your postprocessor.  Three display windows will be presented.  The first window will show you all of the events that were created by the Unigraphics event generator.  The middle window will show you all of the variables that were created and passed to your postprocessor during that event.  The third window is a list of the NC output that was generated by your postprocessor.  If you select any event in the first column, it will show you all of the data generated during that event and the third window will show you the NC code the was generated for that event.  If you select any peice of NC code in the third window, the listing at the top of the display will show you the event that caused the NC code.

The review tool runs after all of the NC code has been generated.  It is not a run time debugger.

The file ugwish.exe must be located in the mach/auxiliary directory for the review tool to work.
