Customization

Post Builder allows you to customization the appearance of the dialogs.  There are two files that may be changed in order to do this.
 

ui_pb_resource.tcl

This file can be modified to change the default font sets, labels, messages and text strings that appear thoughout UG/POST Builder.
Only the system administrator should change this file.  This file is in the POSTBUILD/app/ui directory.

ui_pb_user_resource.tcl

This file can be modified to overwrite the ui_pb_resource.tcl file.  This file can also be found in the POSTBUILD/app/ui directory.This file should be placed in the users home directory in order to personally customize the appearance and behavior of UG/POST Builder for each user.  This file should be copied into the users home directory.

On Unix this directory is /users/log_name.

On Windows this directory is found by typing echo %HOME%  from a DOS command prompt.
 

For example, to change the font appearance on your display, either increase or decrease the size of the numbers in the file.  Only change the value of the numbers or the name of the browser.

##############################################################################
#   UI_PB_USER_RESOURCE.TCL
##############################################################################
#   Description
#   This file defines personalized resources for the PostBuilder.
#
##############################################################################
global gPB env tcl_platform tixOption

#========================
# Font set specification
#========================
if {$tcl_platform(platform) == "windows"} \
{
  set gPB(font)   {ansi 8}
  set gPB(font_sm)  {ansi 6}
  set gPB(bold_font)  {ansi 8 bold}
  set gPB(bold_font_lg)  {ansi 10 bold}
  set gPB(italic_font)  {ansi 8 italic bold}
  set gPB(fixed_font)  {courier 8}
  set gPB(fixed_font_sm) {courier 6}
}
