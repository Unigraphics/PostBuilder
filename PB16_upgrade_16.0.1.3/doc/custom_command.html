Custom Command

This section will allow you to create a Tcl procedure that may be executed from any marker or event.  The system will place all of the correct syntax around the body of your proc.  It will create the first and last line in your proc.  The first line will be:

proc PB_CMD_name_of_proc {} {

and the last line will be

}

This proc will be executed in the sequence that it was placed.  Note that Post Builder will not check the validity of the syntax of custom procs.  This option is for advanced users only.  Post Builder will supply a number of "canned" custom commands that will do pre-determined functions.  Only advanced users should modify any existing custom commands or create their own custom commands.  This option will allow users to capture the edits to the event handler.
